About This Tool

This is a simplified version of the fuzzing.py script from the kaliSung project. The original script uses wordlists from /usr/share/wordlists/dirbuster/ to test thousands of potential directories and files.

Original Script Usage:
python3 ~/hacktools/fuzzing.py
# Tests against testphp.vulnweb.com by default
# Uses DirBuster medium wordlist

Common Directories Tested:
  • /admin - Admin panels
  • /login - Login pages
  • /dashboard - User dashboards
  • /config - Configuration files
  • /backup - Backup files
  • /test - Test environments
  • /robots.txt - Robot exclusions
  • And many more...
