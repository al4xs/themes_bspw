About These Tools

These tools are web-adapted versions of the security utilities included in the original kaliSung BSPWM setup. The original project includes many more command-line tools and system integrations:

Original Command-Line Tools:
  • deepce.sh - Docker enumeration and escapes
  • linpeas.sh - Linux privilege escalation
  • parsing_html.sh - HTML link extraction
  • myfunctions.sh - Shodan queries and recon
  • Various colorscripts - Terminal aesthetics
System Integration Features:
  • BSPWM - Tiling window manager
  • Polybar - Status bar with target display
  • Rofi - Application launcher
  • Firefox Cascade Theme - Minimalist browser
  • Custom Shortcuts - Keyboard-driven workflow
